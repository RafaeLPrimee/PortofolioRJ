Services & Outils Docker

Catalogue complet des services conteneurisés utilisés pour construire une infrastructure cloud complète, flexible et scalable.

🌐 Web & Hébergement
🌐
Apache
Serveur web très populaire.

👉 Sert à héberger des sites web ou applications (HTML, PHP…).
👉 Souvent utilisé avec WordPress ou autres CMS.
🌍
WordPress
CMS très populaire pour créer des sites web.

👉 Permet de faire blogs, sites vitrines, e-commerce (avec plugins).
👉 Fonctionne souvent avec Apache + MySQL.
🗄️ Bases de Données
🗄️
MySQL
Système de gestion de base de données relationnelle.

👉 Stocke les données pour des applications web.
👉 Très courant avec WordPress, GLPI, etc.
🐘
PostgreSQL
Autre système de base de données relationnelle.

👉 Plus avancé que MySQL sur certains points (requêtes complexes, intégrité).
👉 Utilisé par beaucoup d'applications modernes.
🧰
Adminer
Interface web pour gérer des bases de données.

👉 Permet d'accéder à MySQL, PostgreSQL, etc. via navigateur.
👉 Alternative légère à phpMyAdmin.
📊 Gestion de Données (No-Code)
📊
Baserow
Base de données no-code (comme Airtable).

👉 Permet de créer facilement des tables, formulaires et automatisations.
👉 Utile pour gérer des données sans coder.
🧩
NocoDB
Transforme une base SQL en interface type Airtable.

👉 Permet de manipuler des données facilement avec une interface visuelle.
👉 Utile pour équipes non techniques.
📝
AppFlowy
Alternative open source à Notion.

👉 Permet de gérer notes, bases de données, tâches.
👉 Idéal pour organisation personnelle ou travail collaboratif.
🛠️ Administration & Outils IT
🛠️
GLPI
Outil de gestion IT (ITSM).

👉 Gestion des tickets, inventaire matériel, support utilisateur.
👉 Très utilisé par les services informatiques.
🔐
FreeIPA
Solution de gestion des identités et des accès.

👉 Gère utilisateurs, groupes, authentification centralisée (LDAP, Kerberos).
👉 Utilisé dans les environnements d'entreprise.
📈
ProjeQtOr
Outil de gestion de projet open source.

👉 Gère tâches, ressources, planning, budgets.
👉 Adapté aux équipes structurées.
⚙️ Automatisation & Intégrations
⚙️
n8n
Outil d'automatisation (comme Zapier).

👉 Permet de connecter des services et automatiser des workflows.
👉 Exemple : envoyer un mail automatiquement quand un formulaire est rempli.
🐳 Docker & Orchestration
🐳
Portainer
Interface graphique pour gérer Docker.

👉 Permet de créer, surveiller et administrer des conteneurs facilement.
👉 Très pratique pour éviter la ligne de commande.
🔒 Sécurité & Réseau
🔒
PIA (Private Internet Access)
Service VPN.

👉 Permet de sécuriser et anonymiser la connexion internet.
👉 Peut être utilisé dans des conteneurs pour protéger du trafic.

🧠 En Résumé

Web & sites : Apache, WordPress
Bases de données : MySQL, PostgreSQL
No-code data : Baserow, NocoDB
Admin & IT : GLPI, FreeIPA
Automatisation : n8n
Docker : Portainer
Gestion projets : ProjeQtOr
Interface DB : Adminer
Organisation : AppFlowy
Sécurité réseau : PIA
