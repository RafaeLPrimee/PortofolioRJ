Installation VPN avec Certificats

Configuration d'une infrastructure VPN sécurisée utilisant des certificats SSL/TLS pour l'authentification et le chiffrement des connexions.

🔹 1) Prérequis

Tu dois disposer de :

  • 📄 Certificat CA : autorite-certification.p12
  • 🪪 Certificat client : Rafael+JONGERLYNCK+.p12
  • 🌐 Adresse du VPN : vpn.oob.local
  • 🌍 IP publique : 81.255.74.21
  • 🔑 Mot de passe des certificats

🔐 2) Installer les certificats (MMC)

➤ Ouvrir MMC

  • Win + R
  • Tape :
mmc

➤ Ajouter le composant Certificats

  • Fichier → Ajouter/Supprimer un composant logiciel enfichable
  • Certificats → Ajouter
  • Choisir :
    • 👉 Compte d'ordinateur
    • 👉 Ordinateur local

➤ Importer le certificat CA (serveur)

Chemin :

Certificats (ordinateur local)
→ Autorités de certification racines de confiance
→ Certificats
  • Clic droit → Importer
  • Sélectionner : autorite-certification.p12
  • Entrer le mot de passe
  • Terminer

➤ Importer le certificat client (toi)

Chemin :

Certificats (ordinateur local)
→ Personnel
→ Certificats
  • Clic droit → Importer
  • Sélectionner : Rafael+JONGERLYNCK+.p12
  • Entrer le mot de passe
  • Cocher : clé exportable (si proposé)
  • Terminer

✔ Vérification

Ton certificat doit :

  • Avoir une clé privée
  • Contenir : Client Authentication

🌐 3) Résolution du nom VPN

Si vpn.oob.local n'est pas résolu :

Ouvrir en admin :

C:\Windows\System32\drivers\etc\hosts

Ajouter :

81.255.74.21    vpn.oob.local

💻 4) Création du VPN

Ouvrir PowerShell en admin :

Add-VpnConnection `
-Name "VPN-MDS" `
-ServerAddress "vpn.oob.local" `
-TunnelType IKEv2 `
-AuthenticationMethod MachineCertificate `
-EncryptionLevel Required `
-SplitTunneling $False `
-RememberCredential

🔐 5) Configuration IPsec (recommandé)

Set-VpnConnectionIPsecConfiguration `
-ConnectionName "VPN-MDS" `
-AuthenticationTransformConstants SHA256128 `
-CipherTransformConstants AES256 `
-DHGroup Group14 `
-EncryptionMethod AES256 `
-IntegrityCheckMethod SHA256 `
-PFSgroup PFS2048 `
-Force

(optionnel selon version Windows)

🌍 6) Route réseau (optionnel)

Add-VpnConnectionRoute -ConnectionName "VPN-MDS" -DestinationPrefix 192.168.1.0/24

🔄 7) Redémarrage

👉 Redémarrer le PC après :

  • installation certificats
  • modification registre
  • config VPN

🔌 8) Connexion

  • Paramètres → Réseau → VPN
  • Sélectionner VPN-MDS
  • Cliquer sur Connecter

✅ Résultat attendu

  • ✔ Pas de demande login
  • ✔ Connexion automatique avec certificat
  • ✔ Pas d'erreur IKE
  • ✔ Tunnel VPN actif

🚨 Dépannage (si erreur IKE)

Vérifier :

  • Certificat client dans :
    • Ordinateur local → Personnel
  • Certificat CA dans :
    • Autorités de certification racines
  • Nom serveur = vpn.oob.local (pas IP)
  • Certificat contient Client Authentication
  • Clé privée présente
