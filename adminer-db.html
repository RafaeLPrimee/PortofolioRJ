Gestion des Bases de Données

Gestion et administration des bases de données MySQL et PostgreSQL via Adminer, interface web complète pour la gestion de données.

🎯 Objectif

Mettre en place et utiliser Adminer comme interface web pour :

  • Administrer des bases de données MySQL et PostgreSQL
  • Gérer les utilisateurs et les droits
  • Manipuler les données (CRUD)
  • Effectuer des sauvegardes et restaurations

🧰 Prérequis

  • Serveur web (Apache, Nginx ou équivalent)
  • PHP installé (version compatible)
  • Accès à une base MySQL et/ou PostgreSQL
  • Fichier Adminer (adminer.php)

🚀 Installation d'Adminer

  • Télécharger Adminer depuis le site officiel
  • Placer le fichier adminer.php dans le répertoire du serveur web (ex : /var/www/html)

Accéder via navigateur :

http://localhost/adminer.php

🔐 Connexion à une base de données

Étapes :

  • Ouvrir Adminer dans le navigateur
  • Sélectionner le type de SGBD :
    • MySQL
    • PostgreSQL
  • Renseigner :
    • Serveur (ex : localhost)
    • Nom d'utilisateur
    • Mot de passe
    • Base de données (optionnel)
  • Cliquer sur Connexion

🗂️ Gestion des bases de données

5.1 Création d'une base

  • Aller dans Créer une base de données
  • Saisir le nom
  • Choisir l'encodage (UTF-8 recommandé)
  • Valider

5.2 Suppression d'une base

  • Sélectionner la base
  • Cliquer sur Supprimer

⚠️ Attention : action irréversible

📊 Gestion des tables

6.1 Création d'une table

  • Cliquer sur Créer une table
  • Définir :
    • Nom de la table
    • Colonnes (nom, type, taille, contraintes)
  • Valider

6.2 Modification d'une table

  • Sélectionner la table
  • Cliquer sur Modifier la structure
  • Apporter les changements

6.3 Suppression d'une table

  • Cliquer sur Supprimer

📝 Gestion des données (CRUD)

Ajouter des données

  • Cliquer sur la table
  • Sélectionner Insérer
  • Remplir les champs
  • Valider

Lire les données

  • Cliquer sur Sélectionner
  • Appliquer filtres si besoin

Modifier des données

  • Cliquer sur une ligne
  • Modifier les champs
  • Enregistrer

Supprimer des données

  • Cocher les lignes
  • Cliquer sur Supprimer

👥 Gestion des utilisateurs et droits

MySQL

  • Accéder à Privilèges
  • Créer un utilisateur
  • Définir :
    • Mot de passe
    • Droits (SELECT, INSERT, UPDATE, DELETE…)

PostgreSQL

Utiliser requêtes SQL :

CREATE USER utilisateur WITH PASSWORD 'motdepasse';
GRANT ALL PRIVILEGES ON DATABASE nom_base TO utilisateur;

💾 Sauvegarde et restauration

Sauvegarde

  • Cliquer sur Exporter
  • Choisir :
    • Format SQL
    • Tables à exporter
  • Télécharger le fichier

Restauration

  • Cliquer sur Importer
  • Sélectionner le fichier SQL
  • Lancer l'import

🔎 Exécution de requêtes SQL

  • Accéder à Commande SQL
  • Saisir la requête (ex :)
SELECT * FROM utilisateurs;
  • Exécuter

🔒 Bonnes pratiques

  • Utiliser des mots de passe forts
  • Restreindre les accès réseau
  • Sauvegarder régulièrement les bases
  • Éviter l'utilisation du compte root
  • Surveiller les performances et logs

⚠️ Sécurité

  • Ne pas exposer Adminer publiquement sans protection
  • Ajouter une authentification (HTTP ou VPN)
  • Renommer le fichier adminer.php pour plus de sécurité

📌 Conclusion

Adminer est une solution légère et efficace pour gérer MySQL et PostgreSQL via une interface web unique. Il permet d'effectuer toutes les opérations essentielles d'administration sans outils lourds.
